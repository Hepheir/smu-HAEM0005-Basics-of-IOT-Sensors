단일 7-세그먼트 모듈(3611AS)이 1개 밖에 없어서, 4배열 7-세그먼트 모듈인 3641AS로 대체하여 실습하였습니다.

공통 음극방식을 사용하였으머, 각 세그먼트의 GND에 해당하는 핀들의(DIG3, DIG4) 신호를 조절하여 멀티플렉싱하였습니다.
